## General info

CRUD React Typescript App using firebase. You can create book, edit and delete it.

## Technologies

Project created with:

- Vite
- React
- Typescript
- CSS
- Firebase

## Setup

To run this project, install it locally using npm:

```bash
$ cd ../the-librarian2
$ npm install
$ npm run start
```
